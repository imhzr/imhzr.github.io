半步视频爬虫利用you-get批量下载优酷视频

为何叫半步视频爬虫呢，最近在学习爬虫，爬完妹子图后想着爬一下优酷网的电视剧，结果半天没有解析到视频的真正下载地址，本来像借助硕鼠解析正式地址后来发现，硕鼠怂了居然不同优酷的解析了估计是怕被告吧。后来发现you-get这个神器，具体看官网介绍吧，于是只爬取了视频播放页面的url至于解析视频地址和下载就交给you-get吧。

you-get 安装

sudo pip3 install you-get

you-get -i [URL] 查看视频链接信息

给个链接就会自动解析下载，这是太棒了！由于没有找到如何批量下载整个一部电视剧情，我想着在剧情介绍页面把所有剧集的URL爬去下来，然后扔给youget下载就好了。
将剧集列表页面每一集的url爬去下来，保存到dlUrl.txt

dlUrl.txt如下：
http://v.youku.com/v_show/id_XMTY5ODc5Mjg5Ng==.html?from=y1.6-97.3.1.9cd2277647d311e5b692
http://v.youku.com/v_show/id_XMTY5ODc5OTkwNA==.html?from=y1.6-97.3.1.9cd2277647d311e5b692

…..
