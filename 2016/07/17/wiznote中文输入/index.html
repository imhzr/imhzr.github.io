解决wiznote linux 客户端中文输入的问题

参考：http://pkgs.org/fedora-23/fedora-updates-testing-x86_64/fcitx-qt5-devel-1.0.5-3.fc23.x86_64.rpm.html

因为喜欢为知笔记的跨平台性，在fedora下选择了为知笔记，但是安装好为知笔记客户端后，返现居然输入不了中文。

原因：wiznote使用了qt5，而查看我的fcitx安装的是fcitx-qt4，必须安装fcitx-qt5模块

解决办法：
install the fcitx-qt5-devel rpm package:
sudo yum --enablerepo=updates-testing install fcitx-qt5-devel
