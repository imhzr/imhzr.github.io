Python list 函数踩坑小记

最近发现的 python list 函数的小坑，记录备忘。python 版本为 2.7

Python内置的一种数据类型是列表：list。list是一种有序的集合，可以随时添加和删除其中的元素。这个有点类似与java中的数组。
我们知道可以使用索引的方式输出 List 中的元素。（索引从 0 开始，也就是说，第一个元素的索引是0，第二个元素的索引是1，以此类推）
