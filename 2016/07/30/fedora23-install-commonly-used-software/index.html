Fedora23 常用软件安装

1.安装网易云音乐

FZUG.repo，确保里面有开启[fzug-testing]

然后 sudo dnf install netease-cloud-music OK!

参考：https://github.com/FZUG/repo/issues/111

点评：网易良心，linux下的云音乐使用起来跟window下体验一样一样的，非常酸爽！在此小小的鄙视下腾讯的QQ for linux

2.安装 Electronic WeChat

Electronic WeChat是开源爱好者用electron 框架开发的微信客户端可在linux下使用

下载

wget https://repo.fdzh.org/FZUG/testing/24/x86_64/wechat-1.3.0-3.git5058348.fc24.x86_64.rpm

安装

sudo rpm -ivh wechat-1.3.0-3.git5058348.fc24.x86_64.rpm
提示需要安装 electron

安装 electron

sudo dnf install electron

继续安装

sudo rpm -ivh wechat-1.3.0-3.git5058348.fc24.x86_64.rpm
OK！！

3.安装Skype

下载

wget http://download.skype.com/linux/skype-4.3.0.37-fedora.i586.rpm

安装

sudo dnf install skype-4.3.0.37-fedora.i586.rpm
OK ！

4.安装爱壁纸

官网有rpm的包

下载

wget http://s.qdcdn.com/lovebizhi/LoveWallpaper4LinuxFedora.rpm
不过不能直接双击安装，应该很大可能缺少一些依赖导致安装失败
正确的安装方法
sudo dnf install LoveWallpaper4LinuxFedora.rpm
安装过程会下载几个python包安装完成后，可以一键设置壁纸，使用体验跟windows 下一样作为一个爱好壁纸的人来说真是太爽快了
