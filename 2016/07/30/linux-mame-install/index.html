Linux下安装mame街机模拟器

既然要把Linux当做主力机，基本的娱乐肯定是比不可少的嘛。虽然有steame但我更喜欢玩一些街机小游戏，满满的童年回忆啊。鼎鼎大名的街机模拟器mame有linux版。今天正好得空赶紧安装起来玩一把拳皇，哈哈。

1.安裝 mame 模擬器

sudo rpm -Uvh rpmfusion-nonfree-release-23.noarch.rpm （下載地址）
sudo dnf install libspotify-devel
sudo dnf install mame-0.171-1.fc23.x86_64.rpm (下載地址)

2.下載拳皇ROM

需要这两个文件：kof97.zip，neogeo.zip （注：neogeo 的版本不能太低，否則運行會報錯）

参考：http://blog.druggo.org/post/2010/01/11/%E9%80%82%E5%90%88%E6%96%B0%E7%89%88mame%E7%9A%84neogeo-bios

3.運行前準備

mame配置文件位於/etc/mame/mame.ini
查看配置文件可知　roms　位於　/usr/share/mame/roms
sudo mv neogeo.zip kof97.zip /usr/share/mame/roms

4.運行拳皇

mame kof97　注:　此處kof97是rom的名字
如果無意外那麼將會進入拳皇，按tab可以進入菜單設置鍵位,5:投幣,1:ＳTART

額 貌似只能全屏 反正我還沒找到修改的地方,可以修改顯示比例如:4：3 ,16：9 等。但是我沒找到退出全屏的地方

默認 mame 是英文的，應該可以漢化不想折騰了，這點英文還是看得懂的。

好啦，愉快玩耍吧 ！

同理，mame 模擬器的其他遊戲 roms 也可以愉快運行
附：http://coolrom.com/roms/mame/

经测验，拳皇，恐龙快打，合金弹头5等能愉快运行！
